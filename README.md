## Installation

1. Get the core file from the latest released from https://github.com/dead-cells-core-modding/core
2. Unzip it to the game root directory
3. Get the archipelago mods zip from latest released from https://github.com/Maxlamenace572/DeadCellsArchipelago
4. Unzip the mods.zip in the coremod folder

The folder structure should be similar to the following
```txt
<DeadCellsGameRoot>
|
+- coremod
|  |
|  +- core
|  |  |
|  |  +- native
|  |  |  |
|  |  |  +- ...
|  |  |
|  |  +- mdk
|  |  |	 |
|  |  |  +- install.ps1
|  |  |  |
|  |  |  +- uninstall.ps1
|  |  |  |
|  |  |  +- ...
|  |  |
|  |  +- host
|  |  |  |
|  |  |  +- startup
|  |  |  |  |
|  |  |  |  +- DeadCellsModding.exe
|  |  |  |  |
|  |  |  |  +- ...
|  |  |  +- ...
|  |  +- ...
|  +- ...
|
+- deadcells.exe
|
+- deadcells_gl.exe
|
+- ...
```
